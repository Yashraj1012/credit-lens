# Project Screenshots

This directory is reserved for hosting application UI screenshots and assets for the main `README.md` file.

To prepare the repository for a professional release:
1. Run the application locally (FastAPI backend and React frontend).
2. Take clean screenshots of the following views:
   - **Dashboard**: `screenshots/dashboard.png` (displays overall metrics, charts, confusion matrix, global importance)
   - **Risk Assessment**: `screenshots/risk_assessment.png` (displays inputs, risk gauges, borrower risk drivers)
   - **Model Analytics**: `screenshots/model_analytics.png` (displays ROC curve, PR curve, model comparison card)
   - **Developer Diagnostics**: `screenshots/developer_diagnostics.png` (displays collapsible developer panel, tabs showing processed payload)
   - **About Model**: `screenshots/about_model.png` (displays training pipeline flow, API documentation, artifact breakdown)
3. Save the screenshots in this folder using the names listed above.

### Recommended Guidelines
- **Resolution**: Min 1920x1080 for desktop captures.
- **Format**: PNG (compressed) to maintain pixel clarity.
- **Theme**: Take screenshots in both light and dark mode, or standardize on a beautiful dark mode capture.
